# Disciple.Tools Translation Glossary

This glossary provides context and definitions for domain-specific terms used in Disciple.Tools to help translators accurately convey meaning in their language. Use the same translated term consistently throughout for each English term.

---

## Core Concepts

| Term | Description |
|------|-------------|
| **Disciple.Tools** | The name of the software platform. Generally kept as "Disciple.Tools" in translations. |
| **Contact** | An individual person being tracked in the system. This is someone a user is building a relationship with for spiritual purposes. |
| **Record** | A generic term for any data entry in the system (contact, group, etc.). Used when the specific type doesn't matter. |
| **Group** | A gathering of people meeting together, in person or online. Can be at various stages from informal to established church. |
| **Coalition** | A network of organizations or individuals working together toward disciple-making goals. |
| **Disciple Making Movement (DMM)** | A methodology focused on multiplying disciples who make more disciples, often abbreviated as DMM. |

## Users & Roles

### System Roles

| Term | Description |
|------|-------------|
| **Administrator** | System administrator with full access to all settings and data. |
| **Disciple.Tools Admin** | Administrative role specific to Disciple.Tools functionality, may have limited WordPress admin access. |
| **Dispatcher** | A user role responsible for receiving new contacts and assigning them to appropriate Multipliers based on capacity and location. |
| **Multiplier** | A user actively engaged in making disciples. A field worker who receives and follows up with contacts. |
| **Strategist** | A user role focused on planning and strategy for the movement, typically with broader view access. |
| **User Manager** | User role with permission to manage other users and their access levels. |
| **Digital Responder** | A user who responds to inquiries that come through digital/online channels. |
| **Partner** | External organization or individual partnering with the movement; may have limited access. |
| **Registered** | Default user role with basic access. |

### Assignment Relationships

| Term | Description |
|------|-------------|
| **Assigned To** | The primary user responsible for following up with a contact or group. |
| **Sub-assigned** | An additional user helping the primary assignee with a contact. Secondary responsibility. |
| **Accepted** | User has accepted responsibility for an assigned contact. |
| **Decline** | User declines responsibility for an assigned contact. |

### Coaching Relationships

| Term | Description |
|------|-------------|
| **Coach** | Someone who mentors and guides multipliers or other workers. This is spiritual mentoring/discipleship, not sports or business coaching. Consider terms like "mentor" or "guide" if "coach" doesn't translate well in your language. |
| **Coached by** | Indicates who is providing coaching/mentoring to this person. |
| **Is Coaching** | Indicates who this person is coaching/mentoring. |
| **Coached by me** | Filter for contacts that the current user is coaching. |
| **Group Coach / Church Planter** | Person responsible for coaching a group or planting a church. |

## System & Interface

### General Interface Terms

| Term | Description |
|------|-------------|
| **Filter** | A saved set of criteria to narrow down which records are displayed. |
| **Custom Filter** | A user-created filter with specific criteria. |
| **Follow/Following** | Subscribe to receive notifications about changes to a record. |
| **Share** | Give another user access to view and edit a record. |
| **Shared with** | List of users who have access to a record. |
| **Favorite** | Mark a record for quick access; appears in "Favorite" filters. |
| **Starred** | Same as Favorite - marked for quick access. |
| **Assigned to me** | Filter showing only records where the current user is the assignee. |
| **Update Needed** | A flag indicating this record requires attention or follow-up. |
| **Magic Link** | A secure, unique URL that provides access to specific functionality without logging in. |
| **Merge** | Combine two duplicate records into one, preserving data from both. |

| **Bulk Send Message** | Send the same message to multiple contacts simultaneously. |
| **Comments and Activity** | Section on a record showing discussion history and system-logged changes. |
| **Activity** | System-logged changes and events on a record (as opposed to user comments). |
| **Tags** | User-defined labels that can be applied to records for organization. |
| **Fields** | Data entry areas on a record (name, phone, status, etc.). |

| **Contact Info Changed** | Notifications when contact information is updated. |

### Tasks & Reminders

| Term | Description |

## Contacts

### Contact Types

| Term | Description                                                                |
|------|----------------------------------------------------------------------------|
| **Standard Contact** | Regular contact record with normal visibility to assigned users.           |
| **Private Contact** | Contact with restricted access/visibility, only visible to specific users. |
| **Access Contact** | Contact from media or an access strategy.                                  |
| **User Contact** | Contact record linked to a system user account.                            |

### Contact Status

These describe the current state of engagement with a contact.

| Term | Description |
|------|-------------|
| **New Contact** | Contact is new in the system and needs initial processing. |
| **Unassigned** | No user has been assigned responsibility for this contact yet. |
| **Dispatch Needed** | Contact needs to be assigned to a Multiplier by the Dispatcher. |
| **Waiting to be accepted** | Contact has been assigned but the assignee has not yet accepted responsibility. |
| **Active** | The contact is currently being engaged and followed up with. |
| **Paused** | Follow-up is temporarily on hold (see Reason Paused below). |
| **Closed** | The contact relationship has ended (see Reason Closed below). |
| **Archived** | The contact is no longer active but kept for historical records. |
| **Not Ready** | Insufficient information to move forward with the contact at this time. |

### Reason Paused

Sub-status options when a contact is paused.

| Term | Description |
|------|-------------|
| **Contact on vacation** | Contact is temporarily unavailable due to travel. |
| **Contact not responding** | Contact is not engaging with outreach attempts. |
| **Contact not available** | Contact cannot be reached at this time. |
| **Contact has little interest/hunger** | Contact shows low spiritual motivation or interest. |
| **Contact shows no initiative** | Contact lacks proactive engagement in the process. |
| **Contact has questionable motives** | Potential concerns about the contact's intentions. |
| **Ball is in the contact's court** | Waiting for the contact to take the next step or respond. |

### Reason Closed

Sub-status options when a contact is closed or archived.

| Term | Description |
|------|-------------|
| **Duplicate** | Contact is a duplicate of another existing record. |
| **Insufficient contact info** | Cannot maintain contact with inadequate information. |
| **Denies submitting contact request** | Contact denies initiating contact or declines engagement. |
| **Hostile, playing games or self gain** | Contact is not genuinely interested or has ulterior motives. |
| **Only wants to argue or debate** | Contact is argumentative and not receptive to discipleship. |
| **Just wanted media or book** | Contact only sought resources/materials, not relationship. |
| **No longer interested** | Contact has lost interest in continuing. |
| **No longer responding** | Contact has become unresponsive to all outreach. |
| **Already in church or connected** | Contact is already connected with a faith community. |
| **Transferred to partner** | Contact was moved to a partner organization. |
| **Martyred** | Contact was martyred for their faith. This is a sensitive term - use appropriate local terminology. |
| **Moved or relocated** | Contact relocated geographically out of reach. |
| **GDPR request** | Contact requested data deletion per privacy regulations. |

### Contact Sources

Where contacts originated from.

| Term | Description |
|------|-------------|
| **Personal** | Contact from personal relationships or direct ministry. |
| **Web** | Contact from website, online form, or digital outreach. |
| **Phone** | Contact from phone call or SMS. |
| **Facebook** | Contact from Facebook outreach. |
| **Twitter** | Contact from Twitter/X outreach. |
| **Transfer** | Contact transferred from another organization or D.T. site. |
| **Referral** | Contact referred by another person. |
| **Advertisement** | Contact from advertising campaign. |

### Seeker Path

The "Seeker Path" tracks the progression of outreach and relationship building with a contact. These are sequential stages.

| Term | Description |
|------|-------------|
| **Contact Attempt Needed** | Initial state - someone needs to try reaching out to this person. |
| **Contact Attempted** | An attempt was made to reach this person (call, message, visit) but no response yet. |
| **Contact Established** | Two-way communication has been achieved with this person. |
| **First Meeting Scheduled** | An in-person or video meeting has been arranged. |
| **First Meeting Complete** | The first scheduled meeting has taken place. |
| **Ongoing Meetings** | Regular meetings are now happening with this person. |
| **Being Coached** | This person is now receiving intentional discipleship/mentoring. |

### Faith Status

Describes where a person is in their spiritual journey.

| Term | Description |
|------|-------------|
| **Seeker** | Someone exploring faith, asking questions, but has not yet made a personal commitment. Translate descriptively, not pejoratively - this is a neutral term for someone on a spiritual journey. |
| **Believer** | Someone who has made a personal commitment to follow Jesus. Translate descriptively, not pejoratively. |
| **Leader** | A believer who is leading or shepherding others in their faith. |

### Faith Milestones

Significant spiritual steps or practices in a person's journey. These are not necessarily sequential. "Milestones" are markers or significant points along a journey - not requirements or rules. If the journey metaphor doesn't translate well, consider equivalent local metaphors.

| Term | Description |
|------|-------------|
| **Has Bible** | The person possesses a Bible or Scripture in their language. |
| **Reading Bible** | The person is actively reading Scripture. |
| **States Belief** | The person has verbally expressed belief/faith. |
| **Can Share Gospel/Testimony** | The person is capable of explaining the gospel message or sharing their personal story. |
| **Sharing Gospel/Testimony** | The person is actively telling others about their faith. |
| **Baptized** | The person has been baptized. |
| **Baptizing** | The person is baptizing others - a sign of multiplication and leadership. |
| **In Church/Group** | The person is regularly participating in a group of believers. |
| **Starting Churches** | The person is actively involved in starting new groups/churches. |

### Contact Connections

Relationships between contacts.

| Term | Description |
|------|-------------|
| **Connection** | A relationship link between two contacts, often representing spiritual fruit/disciples made. |
| **Baptized by** | The person who baptized this contact. |
| **Baptized** | People this contact has baptized. Shows multiplication. |
| **Coached** | People this contact is coaching/discipling. |
| **Coaching** | Same as "Coached" - people being mentored by this contact. |
| **Subassigned** | Contacts where this person is helping the primary assignee. |

### People Groups

Terms related to tracking ethnic, linguistic, or cultural groups.

| Term | Description |
|------|-------------|
| **People Group** | An ethnic, linguistic, or cultural group. Used to track which populations contacts belong to. |
| **People Groups** | Plural form; the collection of people groups in the system. |
| **People Group Code** | Standardized code identifying the people group (often from external databases). |
| **Primary Language** | The main language spoken by the people group or contact. |
| **Joshua Project** | External database of people groups; D.T. can integrate with Joshua Project data. Keep as proper noun. |

## Groups

### Group Types

Groups progress through stages as they mature.

| Term | Description |
|------|-------------|
| **Pre-Group** | An informal gathering, predominantly of non-believers, that has not yet become a regular group. |
| **Group** | A regular gathering with 3+ believers who do not yet identify as a church. |
| **Church** | A group with 3+ believers that identifies as a church and practices multiple health elements. Consider using terms that convey "gathering of believers" rather than institutional/building concepts, depending on cultural context. |
| **Team** | A group of workers/staff rather than a discipleship group (e.g., leadership team). |

### Group Status

| Term | Description |
|------|-------------|
| **Active** | Group is currently meeting regularly. |
| **Inactive** | Group is no longer meeting. |

### Group Relationships

| Term | Description |
|------|-------------|
| **Parent Group** | The group that founded or birthed this group. Shows multiplication lineage. |
| **Child Group** | A group that has been birthed out of this group. |
| **Peer Group** | Related group that is not in a parent/child relationship - may be collaborating, recently merged, or split. |

### Group Composition

| Term | Description |
|------|-------------|
| **Member Count** | Total number of members in the group. |
| **Member List** | List of contacts who are members of the group. |
| **Leader Count** | Number of identified leaders in the group. |
| **Believer Count** | Number of believers in the group. |
| **Unbeliever Count** | Number of non-believers participating in the group. |
| **Accountable** | Number of people in accountability relationships within the group. |
| **Multiplying** | Number of people actively helping start other groups. |

### Church Health

These are practices or characteristics of a healthy, biblical church. Used to track which elements a group is practicing. "Health" here is a metaphor for vitality and proper function, not physical/medical health.

| Term | Description |
|------|-------------|
| **Baptism** | The group practices water baptism for new believers. |
| **Prayer** | The group regularly prays together. |
| **Praise/Worship** | The group worships together through music, singing, or other expressions. |
| **Fellowship** | Members spend time together building relationships and community. |
| **Communion/Lord's Supper** | The group observes the Lord's Supper/Eucharist together. |
| **Giving** | Members contribute financially or materially to support the group and others. |
| **Sharing the Gospel** | Members actively share their faith with those outside the group. |
| **Scripture/Word** | The group reads and studies the Bible together. |
| **Leaders** | The group has identified and is developing local leadership. |
| **Reproducing** | The group is actively starting or has started new groups. |

## Metrics & Reports

These terms appear in the Metrics section of Disciple.Tools, which provides dashboards and reports for tracking progress.

### Overview & Dashboard Terms

| Term | Description |
|------|-------------|
| **Metrics** | The reporting/analytics section of D.T. showing charts, graphs, and statistics. |
| **Overview** | A summary dashboard showing key statistics at a glance. |
| **My Overview** | Personal dashboard showing the current user's own statistics and progress. |
| **Project Overview** | High-level dashboard showing organization-wide statistics. |
| **Contacts Overview** | Summary view of contact statistics across the system. |
| **Groups Overview** | Summary view of group statistics across the system. |

### Critical Path

The "Critical Path" tracks the overall progress of contacts through the disciple-making process at an organizational level.

| Term | Description |
|------|-------------|
| **Critical Path** | A report showing the progression of contacts through key stages from outreach to multiplication. Visualizes the "funnel" of disciple-making. This is a project management term adapted for ministry - consider whether direct translation, transliteration, or a descriptive phrase works best. |
| **Outreach** | Initial efforts to reach new people - the beginning of the critical path. |
| **Follow Up** | The process of maintaining contact and building relationship after initial outreach. |
| **Movement Tracking** | Monitoring the spread and multiplication of disciples and groups over time. |

### Contact Metrics

| Term | Description |
|------|-------------|
| **New Contacts** | Contacts that have been recently added to the system. |
| **Assigned Contacts** | Contacts that have been assigned to a specific user for follow-up. |
| **Active Contacts** | Contacts currently being engaged (not paused, closed, or archived). |
| **Waiting on Accept** | Contacts assigned to a user who has not yet accepted responsibility for them. |
| **Waiting on Update** | Contacts flagged as needing follow-up or status update from their assigned user. |
| **Stalled** | Contacts where progress has stopped or no recent activity has occurred. |

### Group Metrics

| Term | Description |
|------|-------------|
| **Total Groups** | The complete count of all groups in the system. |
| **Group Types** | Breakdown of groups by their type (pre-group, group, church, team). |
| **Active Group Health Metrics** | Shows which health practices each active group is implementing. |
| **Fully Practicing** | Groups that are practicing all church health elements. |
| **Group and Church Generations** | Visual representation of how groups have multiplied from parent groups. |

### GenMapper & Trees

| Term | Description |
|------|-------------|
| **GenMapper** | A church planting visualization methodology that maps generational relationships. Shows how disciples/groups multiply by tracking who started whom. The name combines "Generation" + "Mapper". Consider whether to transliterate, translate literally, or use a descriptive phrase. |
| **GenMapper Trees** | Visual tree diagrams showing the multiplication/generational relationships between contacts or groups. |
| **Generation** | A level in the multiplication tree. Gen 1 is directly connected to the source, Gen 2 was started by Gen 1, etc. Refers to spiritual lineage, not age. |
| **Node** | A single point/item in a tree diagram representing one contact or group. |
| **Focus** | Zoom in on a specific node to see its descendants in detail. |
| **Data Layers** | Additional information overlaid on tree visualizations (status, dates, etc.). |
| **Highlight Active** | Display option to visually emphasize active records in the tree. |
| **Highlight Churches** | Display option to visually emphasize groups that have become churches. |

### Maps & Locations

| Term | Description |

| **Hover Map** | Interactive map that shows details when hovering over locations. |
| **Locations List** | A text-based list of all locations with associated records. |
| **Regions of Focus** | Geographic areas that users or teams are responsible for. |
| **Location Responsibility** | The geographic areas a user is assigned to cover. |

### Activity & Time-Based Reports

| Term | Description |

| **Activity Highlights** | Summary of significant recent activity. |
| **Daily Activity** | Chart showing activity levels by day. |

| **Time Activity Recorded** | When a specific action or update was logged in the system. |

### Sources & Transfers

| Term | Description |
|------|-------------|
| **Sources** | Where contacts originated from (web form, referral, event, etc.). |
| **Sources Chart** | Visual breakdown of contacts by their source. |
| **Transfer** | Moving a contact from one D.T. site to another (for multi-site networks). |
| **Site Links** | Connections between multiple Disciple.Tools installations for sharing data. |
| **Transferred Contacts** | Contacts that have been sent to or received from partner sites. |

### Four Fields

| Term | Description |
|------|-------------|
| **Four Fields** | A church planting methodology that tracks progress in four areas: Empty Field (outreach), Seeding Field (gospel sharing), Growing Field (discipleship), Harvesting Field (church formation). An optional reporting framework in D.T. Uses agricultural metaphors - if these don't translate well culturally, consider equivalent local metaphors or descriptive alternatives. |
